# Certificate Generator System

A web-based certificate generation system built for workshop and event management.

## How it works
- User scans a QR code
- Landing page opens in browser
- User enters their Name and ID
- A personalized certificate is instantly generated and displayed

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- QR code based access
- Instant certificate generation
- Clean and simple UI
- Works on all devices
